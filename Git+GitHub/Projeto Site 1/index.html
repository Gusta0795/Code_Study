Cachorro Salsicha

De onde vieram?

A história do cachorro salsicha, também conhecido como Dachshund, começa na Alemanha no século XV. O nome significa "cão texugo" em alemão, pois era usado para caçar esse animal.

Como se alimentam?

O cachorro salsicha, também conhecido como Dachshund, deve ser alimentado com ração específica para a raça, porte e idade. É importante fracionar a alimentação em 3 a 4 refeições diárias.
